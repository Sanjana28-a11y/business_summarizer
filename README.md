The *Business Summarizer* is a web-based application built using *Flask* that generates concise summaries from long business documents, reports, articles, or emails.

It provides *two modes of summarization*:

### Abstractive Summarization (BART)
- Generates *new sentences* using deep learning  
- Rewrites content with a high-level understanding  
- Best for professional summaries, reports, and presentations

### Extractive Summarization (TextRank)
- Selects *important original sentences* from the input  
- Preserves exact wording  
- Best for emails, business reports, factual data

---

## Features

- *Two AI summarization modes:* Abstractive (BART) and Extractive (TextRank)
-  Handles long business documents, emails, reports
-  Clean and modern UI using HTML + CSS
-  Fast summarization
-  Simple Flask backend
-  Responsive design
